IOS

There are seven steps to automate IOS configuration management.

  1. Choose an option from IOS menu under IOS to perform a config change
  2. Download Deployment Template which is a spreadsheet to fill in Engineering data
  3. Upload the Template which was downloaded in step 2 with the required Deployment changes
  4. Download Inventory Sample which is a yml file to fill in Administrative data
  5. Upload the Inventory which was downloaded in step 4 with the required Administrative data
  6. Check the files names and click Execute
  7. Verify the Results
