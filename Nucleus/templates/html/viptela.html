Cisco SD-WAN

There are five steps to automate Cisco SD-WAN configuration management.

  1. Choose an option from Cisco SD-WAN menu under Cisco SD-WAN to perform a config change
  2. Download Config Template which is a spreadsheet to fill in Engineering data
  3. Upload the Template which was downloaded in step 2 with the required Config changes
  4. Enter Cisco SD-WAN System Credentials and click Execute
  5. Verify the Results
